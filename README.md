# Banking Application

## Project Overview
This is a Banking Simulator project developed using Spring Boot as part of the Infosys Springboard Internship.

## Features
- Account creation
- Deposit and Withdraw
- Money Transfer
- Block and Unblock Account
- Low Balance Email Alert
- Dashboard using Thymeleaf

## Technologies Used
- Java
- Spring Boot
- Thymeleaf
- Bootstrap
- H2 Database
- Java Mail

## How to Run
1. Open project in Eclipse
2. Run `BankingApplication.java`
3. Open browser and go to:
   http://localhost:8080/accounts/dashboard
